# Universal StickyNotes

### imagine a paperclip company developing an AI by giving it the simple task of creating paperclips. It'll eventually spiral out of control leading to exponential but unethical efficiency of simply making paperclips. This is the Paperclip Dilemma. It was a thought experiment by Nick Bostrom. I decided that I want to make a 3D tycoon-esque game based off of Universal Paperclips, but instead with Sticky Notes.
### You can read more about it [here](https://en.wikipedia.org/wiki/Instrumental_convergence)
